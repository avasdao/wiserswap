# WiserSwap

### https://wiserswap.com

[![codecov](https://codecov.io/gh/avasdao/wiserswap/branch/master/graph/badge.svg)](https://codecov.io/gh/avasdao/wiserswap)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

> Automated Market Maker (AMM) built to support over 100K asset transfers per second (TPS) on [Nexa's](https://nexa.org) innovative blockchain network.

## Features

- **Trustless & Permissionless** — "Wise" contracts power every swap with zero counterparty risk
- **100K TPS Scalability** — Blazing fast L1 ensures the lowest fees in crypto
- **Global "Shared" Liquidity** — Hundreds of pools satisfy even the most demanding traders
- **Non-Custodial** — Your Keys. Your Coins. Assets remain 100% under your control
- **Real-time Market Data** — Live ticker information to maximize your competitive advantage

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Run tests
pnpm test

# Run tests with coverage
pnpm test:coverage
```

## Tech Stack

- [Nuxt 3](https://nuxt.com) — Vue 3 full-stack framework
- [Pinia](https://pinia.vuejs.org) — State management
- [Tailwind CSS](https://tailwindcss.com) — Utility-first CSS
- [NexaJS](https://nexajs.org) — Nexa blockchain SDK
- [Vitest](https://vitest.dev) — Unit testing framework
